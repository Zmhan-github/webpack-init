# webpack-init

Это стартовая сборка webpack v4   


```
git clone https://github.com/Zmhan-github/webpack-init.git

npm install

npm run build
```

go to **public/index.html**


В сборку входит:

ES6 -> ES5

SASS -> CSS

IMG -> public/img

Структура public сборки:

```
.
├── bundle.js
├── favicon.ico
├── img
│   └── logo.svg
├── css
│   └── styles.css
└── index.html

1 directory, 4 files
```
